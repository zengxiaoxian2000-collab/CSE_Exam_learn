# 中级注册安全工程师《法律法规》命题逻辑库 (Exam DNA)

> 本文件基于2018-2024年7年真题的深度剖析，提取命题规律、陷阱设置模式和考点关联逻辑。作为后续出题的绝对准则。

---

## 一、陷阱设置规律

### 1.1 主语偷换（最高频陷阱，占陷阱题量约60%）

命题人将正确条文中的主语替换为错误主语，是注安考试最核心的陷阱手法。

#### 1.1.1 责任主体偷换

| 偷换维度 | 常见混淆对 | 示例 |
|---------|-----------|------|
| 企业内角色 | 主要负责人 vs 安全生产管理人员 vs 从业人员 | “安全管理人员的职责”偷换为“主要负责人的职责” |
| 企业间角色 | 建设单位 vs 施工单位 vs 设计单位 vs 监理单位 | 安全设施设计审查责任主体偷换 |
| 政府层级 | 县级 vs 市级 vs 省级 vs 国务院部门 | 审批/备案/考核的管辖层级偷换 |
| 党政角色 | 应急管理部门 vs 公安机关 vs 行业主管部门 | 监管主体偷换（如爆破作业由公安监管，不是应急） |

**典型真题示例（2023年）：**
- 题目问“安全生产管理人员的职责”，选项混入“保证安全生产投入的有效实施”（这是主要负责人的职责）
- 题目问“企业隐患排查治理责任主体”，选项出现“安全生产管理机构是责任主体”（应为“生产经营单位”）

#### 1.1.2 主语偷换的固定模板

```
模板1: 将【单位】的职责偷换给【个人】
  例: "生产经营单位是隐患排查治理的责任主体" → 偷换为 "主要负责人是..."

模板2: 将【A部门】的职责偷换给【B部门】
  例: "应急管理部门负责..." → 偷换为 "公安机关负责..."

模板3: 将【A单位】的职责偷换给【B单位】
  例: "建设单位应当..." → 偷换为 "施工单位应当..."

模板4: 将【发包方】的职责偷换给【承包方】
  例: "生产经营单位对承包单位统一协调管理" → 偷换为 "承包单位对自身统一管理"
```

### 1.2 数字游戏（第二大陷阱类型）

命题人通过修改数字阈值来制造错误选项，考查对精确数字的记忆。

#### 1.2.1 数字陷阱分类

**A. 人员数量阈值（> X 人 / < X 人）**
| 数字 | 关联规则 |
|------|---------|
| 100人 | 安全管理机构/专职安全管理人员配备分界线 |
| 300人 | 注册安全工程师按15%比例配备的基数要求 |
| 7人 | 安全管理人员不足7人至少配1名注安师 |
| 1000人/5000人/5000人以上 | 大型群众性活动安全许可分级 |

**B. 时间期限（极易混淆）**
| 时间 | 关联规则 |
|------|---------|
| 30日 | 事故伤亡补报（道路/火灾为7日）、工伤认定申请、特种设备使用登记、安全设施设计审查 |
| 60日 | 特种作业操作证复审申请、工伤认定决定 |
| 15日 | 事故调查报告批复、行政处罚听证申请（5日内）|
| 7日 | 听证通知、道路/火灾事故补报期限 |
| 24小时 | 停止供电提前通知 |
| 3个月/半年/1年/2年/3年 | 应急预案演练频次 |

**C. 罚款金额（上下限极易记混）**
| 事故等级 | 对单位罚款 | 对主要负责人罚款（上年收入%） |
|---------|-----------|--------------------------|
| 一般事故 | 30万-100万 | 40% |
| 较大事故 | 100万-200万 | 60% |
| 重大事故 | 200万-1000万 | 80% |
| 特别重大事故 | 1000万-2000万 | 100% |

**D. 培训学时**
| 人员类型 | 初次培训 | 每年再培训 |
|---------|---------|-----------|
| 一般单位主要负责人/安管人员 | 不少于32学时 | 不少于12学时 |
| 高危行业主要负责人/安管人员 | 不少于48学时 | 不少于16学时 |
| 新上岗从业人员 | 不少于24学时 | - |

**E. 比例类**
| 比例 | 关联规则 |
|------|---------|
| 15% | 注册安全工程师配置最低比例 |
| 60% | 主要负责人/安管人员对谎报瞒报事故的罚款（上年收入的60%-100%） |
| 50% | 继续教育专业课程应不少于总学时的一半 |

#### 1.2.2 数字游戏的固定模板

```
模板1: 变更上限数字
  例: "处2万元以上5万元以下罚款" → 改为 "处5万元以上10万元以下罚款"

模板2: 变更期限天数
  例: "期满前60日内申请复审" → 改为 "期满前30日内"  (正确: 特种作业操作证复审需在期满前60日)

模板3: 变更人员阈值
  例: "从业人员超过100人应设安全管理机构" → 改为 "50人"

模板4: 混淆同类数字
  例: 将"应急预案至少每半年演练1次"改为"每季度1次"，或把高危/一般企业的培训学时互调
```

### 1.3 关键词替换陷阱

命题人将法律条文中的核心动词或名词替换为近义词，看似相似实则错误。

| 正确表述 | 错误替换 | 语境 |
|---------|---------|------|
| 应当 | 可以 | 强制性条文被弱化 |
| 不得 | 可以/应当 | 禁止性条文被篡改 |
| 审查 | 备案/告知 | 行政审批强度被降级 |
| 论证 | 评审 | 小型企业是"论证"，中型+是"评审" |
| 专职 | 兼职 | 人员配备性质被改变 |
| 查封 | 扣押 | 强制措施对象错位 |
| 停产停业 | 停止使用 | 行政处罚种类混淆 |
| 立即 | 限期 | 处置时效被降级 |
| 设区的市级 | 县级/省级 | 政府层级被改变 |
| 安全生产监督管理部门 | 应急管理部门 | 旧法条用旧名称（注：2021年修正后改为应急管理部门） |

### 1.4 例外排除陷阱

命题人将法规的“不适用”或“除外”情形作为考点，极容易出错。

**高频考点示例：**
- 《危险化学品安全管理条例》**不适用**：民用爆炸物品、烟花爆竹、放射性物品、核能物质
- 《特种设备安全法》**不适用**：军事装备、核设施、航空航天器、铁路机车、海上设施和船舶、矿山井下特种设备、民用机场专用设备
- 《危险化学品输送管道安全管理规定》**不适用**：原油/成品油/天然气长输管道、城镇燃气管道的安全管理
- 工伤认定中**不算工伤**：故意犯罪、醉酒或吸毒、自残或自杀
- 不得进入高速公路：行人、非机动车、拖拉机、全挂拖斗车、设计最高时速低于70公里的机动车

### 1.5 绝对化表述陷阱

选项中若使用绝对化词汇，多为错误选项：
- "一律"、"任何情况下都"、"必须"、"全部"、"只有...才"
- 但注意：有少数条目确实使用"不得""禁止""严禁"等绝对词，需要具体判断

## 二、场景化命题模式

### 2.1 场景分类体系

命题人将法律条文包装在具体企业/作业场景中，考查法规的实际适用能力。

#### 2.1.1 企业类型场景（高频）

| 场景类型 | 标签 | 考点触发 |
|---------|------|---------|
| 危险化学品生产企业 | 危化-生产 | 安全使用许可证、安全评价、应急预案备案 |
| 危险化学品经营/储存企业 | 危化-经营 | 经营许可证、储存要求、登记制度 |
| 矿山企业（煤矿/非煤矿山） | 矿山 | 安全生产许可证、安全设施设计审查、领导带班下井 |
| 建筑施工企业 | 建筑 | 安全生产许可证、专项施工方案、交叉作业 |
| 金属冶炼企业 | 冶金 | 重大事故隐患判定、应急预案评审 |
| 烟花爆竹企业 | 烟花 | 批发/零售许可、燃放规定 |
| 民用爆炸物品企业 | 民爆 | 销售/购买/运输许可 |
| 交通运输企业 | 交通 | 道路运输安全、危险货物运输 |
| 工贸企业（粉尘涉爆/有限空间） | 工贸 | 粉尘防爆、有限空间作业 |
| 人员密集场所（商场/体育场/景区） | 人员密集 | 消防安全、大型活动安全管理 |
| 食品/纺织/机械等一般企业 | 一般工业 | 安全管理机构配备分界线、培训学时 |

#### 2.1.2 作业类型场景

| 场景 | 触发法规 |
|------|---------|
| 动火作业 | 《安全生产法》第43条 危险作业现场管理 |
| 爆破作业 | 《民用爆炸物品安全管理条例》 |
| 吊装作业 | 《安全生产法》危险作业管理 |
| 临时用电 | 《安全生产法》危险作业 + 施工现场临时用电 |
| 有限空间作业 | 《工贸企业有限空间作业安全管理与监督暂行规定》 |
| 高处作业 | 《特种作业人员安全技术培训考核管理规定》 |
| 交叉作业 | 《安全生产法》第48条 签订安全管理协议 + 专职安全管理人员 |
| 拆除作业 | 《建筑法》《建设工程安全生产管理条例》 |
| 粉尘爆炸危险场所作业 | 《工贸企业粉尘防爆安全规定》 |
| 外包/发包/出租 | 《安全生产法》第49条 统一协调管理 |

#### 2.1.3 事故场景

| 事故类型 | 考点触发 |
|---------|---------|
| 火灾事故 | 消防法、应急预案、事故报告 |
| 爆炸事故 | 危化品条例、民爆条例、烟花爆竹条例 |
| 坍塌事故 | 建筑法、建设工程安全生产管理条例 |
| 中毒窒息 | 有限空间作业规定、职业病防治法 |
| 交通事故 | 道路交通安全法、事故报告补报(7日) |
| 泄漏事故 | 危化品条例、突发事件应对法 |
| 高处坠落 | 特种作业人员规定、劳动防护 |

### 2.2 场景嵌套模式（难点所在）

命题人常用**多层场景嵌套**来增加难度：

```
嵌套模板1: 企业类型 + 作业类型 + 法律责任
  例: "某化工企业（危化品生产）进行动火作业（危险作业），未安排专门人员现场管理，
       发生事故造成2人死亡（一般事故），对单位和主要负责人的处罚是？"

嵌套模板2: 企业类型 + 人员身份 + 资格要求
  例: "某矿山企业（高危）的安全生产管理人员（特定角色），
       初次安全培训时间不得少于？"

嵌套模板3: 行政区划 + 企业类型 + 审批层级
  例: "某县的危险化学品生产企业（特定行业），
       其安全设施设计审查应当向哪一级应急管理部门提出？"
```

## 三、多考点关联性

### 3.1 考点关联图谱

命题人常在一个题目中串联多个相关考点，形成“考点链”。

#### 3.1.1 事故等级 → 罚款 → 刑事责任 → 资格限制

```
【考点链】
事故等级判定（死亡人数/重伤人数/直接经济损失）
  → 对单位的罚款金额（30万-2000万）
    → 对主要负责人的罚款（年收入40%-100%）
      → 刑事责任（3年以下/3-7年/7年以上）
        → 职业禁入年限（5年/终身）
```

#### 3.1.2 企业类型 → 安全管理机构 → 人员配备 → 培训学时

```
【考点链】
企业是否属于高危行业（矿山/危化/建筑/运输/冶金）
  → 需要安全管理机构还是可以仅配备人员
    → 配备专职还是可以兼职
      → 是否需要注册安全工程师（15%比例）
        → 初次培训和每年再培训学时
```

#### 3.1.3 行业类型 → 行政许可 → 审批部门层级

```
【考点链】
行业类别（危化品/矿山/建筑/烟花/民爆）
  → 需要什么许可证（安全生产许可证/经营许可证/安全使用许可证）
    → 向哪一级部门申请（县级/市级/省级）
      → 审批时限
        → 许可证有效期和延续
```

#### 3.1.4 应急预案 → 编制 → 评审/论证 → 演练 → 备案

```
【考点链】
企业类型（高危/中型+/小型）
  → 应急预案类型（综合/专项/现场处置方案）
    → 是否需要评审（高危+中型以上）/论证（其他）
      → 演练频次（高危半年一次/政府应急管理部门两年一次）
        → 备案部门（县级以上应急管理部门）
```

### 3.2 横向关联（跨法规联动）

命题人会将同一事项在多个法规中的规定串联，考查区分能力。

| 关联主题 | 涉及法规 | 区分要点 |
|---------|---------|---------|
| 特种作业操作证 | 《安全生产法》《特种作业人员安全技术培训考核管理规定》 | 证书种类、复审期限(60日)、考核发证机关 |
| 安全评价/检测检验 | 《安全生产法》《安全评价检测检验机构管理办法》 | 虚假报告罚款、资质要求 |
| 事故报告 | 《安全生产法》《生产安全事故报告和调查处理条例》《生产安全事故信息报告和处置办法》 | 报告时限(1小时/立即)、报告对象(应急管理部门)、补报期限(30日/7日) |
| 应急救援 | 《安全生产法》《生产安全事故应急条例》《生产安全事故应急预案管理办法》 | 应急救援组织建立义务、预案编制与备案 |
| 危险化学品 | 《危险化学品安全管理条例》《危险化学品重大危险源监督管理暂行规定》《危险化学品输送管道安全管理规定》 | 适用范围区分、备案/审查层级 |
| 消防安全 | 《消防法》《建设工程消防设计审查验收管理暂行规定》《高层民用建筑消防安全管理规定》 | 审查/验收/备案分类、专职消防队建立条件 |

### 3.3 区分型关联（最易出错）

两个高度相似但规则不同的事项被命题人放在同一题目中对比。

| 对比对 | 区分点 |
|--------|-------|
| 主要负责人 vs 安全生产管理人员 | 职责完全不同，前者“建立/组织/保证/实施”，后者“组织或参与/检查/督促/制止” |
| 矿山/危化等高危行业 vs 一般行业 | 培训学时(48/16 vs 32/12)、安全管理机构设置、注安师配备要求 |
| 应急预案 评审 vs 论证 | 高危+中型的其他单位需要"评审"，小型/微型可以"论证" |
| 特种设备 使用登记 vs 定期检验 | 登记是使用前/后30日内，检验是有效期满前一个月提出 |
| 行政处罚 听证vs 行政复议 | 听证在告知后5日内提出，7日前通知，期限完全不同 |
| 安全生产许可证 vs 安全使用许可证 vs 经营许可证 | 不同行业适用不同许可证，颁发部门也不同 |

## 四、命题技巧汇总

### 4.1 选项设计规律

**单选题：**
- 4个选项中，1个完全符合法条原文（正确答案），2个为主语偷换型错误，1个为数字修改型错误
- 常有2个选项表述高度相似，仅在1-2个关键词不同（如“专职”vs“兼职”、“审查”vs“备案”）
- 正确选项常使用法条原文表述，不使用绝对化语言

**多选题：**
- 5个选项中，通常2-4个正确
- 常见为3个正确（3/5模式）或2个正确（2/5模式）
- 错误选项通常是主语偷换或绝对化表述
- 完全陌生/冷门的表述通常不选

### 4.2 干扰项制造手法

1. **张冠李戴**：将A法规的规定套用到B法规的情境中
2. **数字移位**：将正确数字增减（如"30日"变"15日"）
3. **层级错位**：将审批部门从"市级"改为"县级"
4. **时间倒置**："前"变"后"、"内"变"外"
5. **性质混淆**："强制性"变"推荐性"、"审查"变"备案"
6. **范围扩大/缩小**："剧毒化学品"变"危险化学品"（扩大了范围）

### 4.3 正确答案特征

1. **语义中庸**：不含绝对化词汇（但禁止性条款除外）
2. **表述严谨**：与法条原文高度一致
3. **条件充分**：包含必要的限定条件（如"设区的市级"而非仅"市级"）
4. **逻辑自洽**：不合常识的极端表述通常错误

## 五、出题绝对准则

### 5.1 必须遵守的原则

1. **每道题必须对应至少一个法条原文出处**（《安全生产法》《危险化学品安全管理条例》等核心法规）
2. **干扰项必须基于真实易混淆点**，不得生造无意义的错误表述
3. **场景设置必须符合行业真实情况**（企业类型、作业场景、监管部门等）
4. **数字考点必须精准**，正确选项数字与法条原文一字不差
5. **多选题应至少有2个且不超过4个正确选项**
6. **题目难度梯度**：60%为单一考点直选题，30%为两考点关联题，10%为三考点以上综合题

### 5.2 陷阱设置优先级

根据历年真题统计，陷阱使用频率为：

1. 主语偷换（60%）- 最优先使用
2. 数字修改（20%）- 次优先使用
3. 关键词替换（10%）- 适度使用
4. 例外排除（5%）- 点缀使用
5. 绝对化表述（5%）- 少用，仅对非禁止性条款使用

### 5.3 场景使用优先级

根据命题频率，优先使用以下场景：

1. 危险化学品企业场景（30%的题目涉及）

3. 矿山/煤矿场景（15%）
4. 金属冶炼/工贸场景（10%）
5. 人员密集场所/消防场景（10%）
6. 道路交通运输场景（5%）
7. 其他综合场景（10%）

## 六、真题验证示例

以下是真题中陷阱设置的经典案例验证以上规律：

**案例1（2023年第1题 - 主语偷换 + 关键词替换）：**
题干问“安全生产管理人员的职责”，选项中出现“保证本公司安全生产投入的有效实施”——这是主要负责人的职责，典型的主语偷换。

**案例2（2023年第13题 - 数字游戏）：**
“王某2022年被聘为危险化学品生产企业总经理...2023年参加安全培训的最少时间是？”
正确：高危行业主要负责人初次培训48学时，每年再培训16学时 → 答案16学时
干扰：将48和32、16和12混淆（分别对应高危和一般行业）

**案例3（2024年第6题 - 企业类型 + 人员配置关联）：**
不同企业类型（航空运输60人/制衣80人/甲醇销售20人/机械加工110人）对应不同的安全管理机构/人员配备要求，考查企业分类+人员阈值+专兼职区别三个维度的关联。

**案例4（2023年第12题 - 多层嵌套）：**
“甲公司是乙县丙镇的氯乙烯生产企业”（企业类型+行政区划），考查应急预案演练频次（高危每半年）、演练情况报送对象（县应急管理部门）、政府预案演练频次（每2年）——在一个题目中综合考了多个关联考点。
